== Description ==
 
Epic Block Slider for Gutenberg adds image slider blocks you need to build professional websites. Build in Crop tool to help make fit images into the slider perfectly. Custom text and button blocks that have the ability to add animation to help grab the attention for your visitors.
 
== Plugin Features ==

- adds 5 Image slider to Gutenberg blocks.
- Include custom text and button block that add animition features when show on slide in.
- Include the ablity to crop image to to match size of to all image for seamless look.

== Installation ==

1.  Upload your plugin folder to the '/wp-content/plugins' directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
